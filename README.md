# Python-Autograder-Template
This is a template autograder written in Python that works with Gradescope.


Your folder structure should look like this.<br>
- autoGraderRootDir<br>
  - source<br>
    - tests<br>
	  - test_subprocess.py<br>
	  - utils.py<br>
	- run_autograder<br>
	- run_tests.py<br>
	- setup.sh<br>
  - submission<br>
	- mysubmission.c<br>
    - makefile<br>
